## Overview
This project is a low-cost non-invasive anemia detection system that estimates hemoglobin levels using an IR sensor and Arduino without requiring a blood sample.

## Features
- Non-invasive measurement
- Low-cost hardware
- Quick results
- Portable design

## Components Used
- Arduino Uno
- IR Sensor Module
- 16x2 LCD Display
- Breadboard
- Jumper Wires

## Working Principle
IR light passes through the finger and is received by the IR sensor. The transmitted light intensity is used to estimate hemoglobin level.
